Data binding
Introduction to data binding. Data binding is an important part of the angular application. It allows really dynamic flow of data.
String interpolation

The value '{{ property1 }}' is bound to a variable in the typescript file

Property binding

The 'href' property of this link is bound to a variable in the typescript file

Event binding

The 'click' event of this is bound to a function in the typescript file

Surprise!

The visibility of this item is controlled by a boolean 'showStuff' in the typescript file

Two way binding

It is possible to bind both ways, this can be useful with input fields. Two way binding basically listens to the key presses and updates the underlying property when typing and at the same time binds it back to the input field.
