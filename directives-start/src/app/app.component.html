Directives
Introduction to directives. Directives can be really complex, but they are basically 'components without a template'. Directives are designed to modify the DOM. There are 2 types of directives:
  • Attribute directives: Modify the attributes of an element, for example [ngStyle] is an attribute directive that changes the 'style' of an HTML element
  • Structural directives: Modify the actual DOM, for example *ngFor is a structural directive.
Important: the * (asterisk) is angular syntax, the * just means that angular will create necessary templates and hooks so that it knows where this element is/was or would be when the DOM is modified.
